# Apache NiFi

## Getting Started

1. Download: https://archive.apache.org/dist/nifi/2.8.0/nifi-2.8.0-bin.zip
2. Requirement: Java 21+
3. Set username and password: ./nifi/nifi-2.8.0-bin/nifi-2.8.0/bin/nifi.sh set-single-user-credentials fhofer fsu-nifi-tool
4. Start NiFi: ./nifi/nifi-2.8.0-bin/nifi-2.8.0/bin/nifi.sh start
5. Go to: https://localhost:8443/nifi
